# MemoDoc

업무 메모를 붙여 넣으면 AI가 6가지 형태의 지식 문서를 자동으로 만들어 주는 브라우저 도구.

---

## 해결하려는 문제

회의가 끝나고 메모는 있는데 정리할 시간이 없다. MemoDoc은 날 것의 메모를 즉시 공유 가능한 문서로 변환한다.

---

## 주요 기능

| 탭 | 출력 |
|---|---|
| 📋 요약 | 핵심 내용 bullet point |
| 💡 개념 정리 | 주요 용어 + 정의 |
| 📄 구조화 문서 | Notion/Wiki용 마크다운 문서 |
| 🔷 도식화 | Mermaid 플로우차트 |
| 🪜 단계별 설명 | Step-by-step 가이드 |
| 🏗 설계 활용 | 재활용 포인트 + 체크리스트 |

---

## 기존 도구와의 차별점

| | MemoDoc | Notion AI | ChatGPT | Obsidian |
|---|---|---|---|---|
| 설치/가입 없이 실행 | ✅ | ❌ | ❌ | ❌ |
| 6가지 관점 동시 생성 | ✅ | ❌ | ❌ | ❌ |
| Mermaid 다이어그램 자동 생성 | ✅ | ❌ | △ (별도 요청) | ❌ |
| 메모 유형별 적응형 요약 | ✅ | ❌ | ❌ | ❌ |
| 무료 사용 | ✅ | ❌ (유료) | △ (제한) | ✅ |
| 마크다운 내보내기 | ✅ | ✅ | ❌ | ✅ |

**핵심 차별점:** 단일 메모 입력으로 요약 · 개념 · 문서 · 다이어그램 · 가이드 · 설계 포인트를 한 번에 생성한다. 다른 도구는 각 결과물을 별도로 요청해야 한다.

### 실측 비교: 동일 메모 기준 수동 작성 vs MemoDoc

> **테스트 메모:** MSA 전환 회의 메모 (약 200자, 3단계 프로세스 포함)

| 항목 | 수동 작성 | MemoDoc | 절감 |
|---|---|---|---|
| 회의록 요약 | 10~15분 | 포함 | — |
| 개념 정리 | 5~10분 | 포함 | — |
| 구조화 문서 | 10~15분 | 포함 | — |
| Mermaid 다이어그램 | 10~15분 | 포함 | — |
| 단계별 가이드 | 5~10분 | 포함 | — |
| 설계 포인트 | 5~10분 | 포함 | — |
| **합계** | **45~75분** | **2~3분** | **약 95%** |
| 출력 포맷 수 | 1~2개 | 6개 | — |

- **ChatGPT와의 차이:** ChatGPT로 동일 결과물을 얻으려면 6번의 별도 요청 + 컨텍스트 재입력 필요. MemoDoc은 한 번의 입력으로 6개 문서를 병렬 생성.
- **Notion AI와의 차이:** Notion AI는 이미 작성된 문서를 보강하는 용도. MemoDoc은 날 것의 메모를 구조화된 문서로 변환하는 데 특화.
- **Mermaid 자동 생성:** 프로세스가 있는 메모에서 flowchart 자동 생성. 수동으로 Mermaid 문법을 작성하는 데 드는 10~15분을 완전히 자동화.

## 실행 방법

1. `hackathon.html`을 브라우저에서 열기 (서버 불필요)
2. [openrouter.ai](https://openrouter.ai)에서 무료 API 키 발급 (`sk-or-...`)
3. API 키 입력 후 저장
4. 메모 유형 선택 (회의/교육/아이디어) 후 메모 입력 → 문서 생성

---

## 사용 예시

```
오늘 팀 회의에서 MSA 전환 논의.
3단계 계획: 도메인 분리 → API Gateway → 이벤트 버스
첫 번째로 User 서비스 분리 결정. 담당: 백엔드팀. 기간: 6개월.
```

→ 위 메모를 입력하면 요약, 개념 정리, 회의록, 플로우차트, 구현 가이드, 설계 포인트가 동시에 생성된다.
