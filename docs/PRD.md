# MemoDoc - PRD

> 작성일: 2026-03-13 | 버전: 1.0

---

## 배경

업무 중 작성한 메모는 대부분 개인 노트에서 멈춘다. 나중에 꺼내보면 맥락 없이 단어만 나열되어 있고, 팀에 공유하거나 설계에 재활용하기 어렵다.

---

## 문제 정의

- 회의 메모를 공식 문서로 정리하는 데 시간이 너무 걸린다
- 다이어그램이나 플로우차트 작성 진입 장벽이 높다
- 메모가 지식 자산으로 이어지지 않는다

---

## 목표

업무 메모를 붙여 넣으면 AI가 즉시 6가지 형태의 지식 문서를 자동 생성해 준다.

---

## 사용자

소프트웨어 개발자 / 기획자. 회의 직후, 기술 학습 후, 설계 작업 전 사용.

---

## 기능 요구사항

| ID | 기능 | 설명 |
|---|---|---|
| FR-1 | 메모 입력 | 자유 형식 텍스트 입력 (최대 10,000자) |
| FR-2 | 요약 | 핵심 내용 bullet point 정리 |
| FR-3 | 개념 정리 | 주요 용어 추출 + 정의 |
| FR-4 | 구조화 문서 | 마크다운 공식 문서 생성 |
| FR-5 | 도식화 | Mermaid 플로우차트 자동 생성 |
| FR-6 | 단계별 설명 | Step-by-step 가이드 |
| FR-7 | 설계 활용 포인트 | 재활용 가이드 및 체크리스트 |
| FR-8 | 내보내기 | .md 파일 다운로드 (탭별 / 전체) |

---

## 범위

**포함:** 브라우저 단독 실행, AI 문서 생성 6종, 마크다운 렌더링, 다이어그램 렌더링, 내보내기, 다크모드

**미포함:** 서버, 로그인, 문서 저장/히스토리, 팀 공유

---

## 사용자 시나리오

1. 브라우저에서 `hackathon.html` 열기
2. OpenRouter API 키 입력 후 저장
3. 회의 메모를 텍스트 영역에 붙여 넣기
4. "문서 생성" 클릭 → 6개 탭이 동시에 스트리밍으로 채워짐
5. 원하는 탭의 결과를 .md로 내보내기

---

## 성공 기준

- 메모 입력 후 3초 이내 첫 텍스트 출력 시작
- 6개 탭 모두 오류 없이 결과 생성
- Mermaid 다이어그램 정상 렌더링
- 내보낸 .md 파일이 Notion/GitHub에서 그대로 사용 가능

---

## MVP 범위

해커톤 완료 기준: `hackathon.html` 하나로 실행 → API 키 입력 → 메모 입력 → 6탭 문서 생성 → 내보내기까지 전 흐름 동작.
